# joomla-ansible

The project consist in a group of roles for fully automized deploy Joomla CMS with Ansible. The roles has been tested on Ubuntu 18.04 Amazon EC2 istances.

Directory trees:

roles
    ├── galera_cluster
    │   ├── handlers
    │   │   └── main.yml
    │   ├── README
    │   ├── tasks
    │   │   └── main.yml




Roles:
- upgrade
- nginx
- nginx-lb
- mysql
- php
- joomla
- galera_cluste
- galera_node-
